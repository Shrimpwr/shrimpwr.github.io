失落的藏宝图

提示“笑迎八方客 喜得四面财”，是四方密码
四方密码需要两个密钥，第一个密钥，看“Uspaiimsdevcfvl! Rzi hciam iwc km jmvsjmvs, ewp eg xq zzvw rzi uytwgb grg.”这行，是维吉尼亚密码，且可以看出第一个单词是 “Congratulations”，解出“Congratulations! The first key is security. now go to find the second one.”
第二个密钥，给出了一行像是乱码的汉字“亠鯛顼魻翳鍫箦泔钿犽妁澡鍫脲鶢轶犻铈矧磲糸镱”（中间的不是空格），用winhex打开，复制出16进制数据，逐个减去128，再转成字符输出，得到 “You got the second key! The key is information”
解密得到flag{welcometochive}
